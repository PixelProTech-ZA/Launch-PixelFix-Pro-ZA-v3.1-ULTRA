## FEATURES

### MODULE 01 — Edit & Resize
- Upload via drag & drop or tap
- 8 built-in SA document presets
- SA ID · Passport · Visa · Square · WhatsApp · LinkedIn · Instagram · FB Cover
- Custom width × height with aspect ratio lock
- DPI control (72 web / 300 print / 600 hi-res)
- Output format: JPG · PNG · WebP
- Quality slider
- **5 enhancement sliders:** Sharpen · Radius · Brightness · Contrast · Saturation
- Background colour — 7 swatches + custom colour picker
- Auto center crop or letterbox mode
- Animated processing overlay with dual-ring spinner
- Export flash confirmation on download

### MODULE 02 — Background Remover
- Edge-detection background removal algorithm
- Adjustable sensitivity threshold
- Replace background with: Transparent · Solid colour · Custom uploaded image
- 7 preset colours including SA ID blue
- Always exports as PNG to preserve transparency
- 100% local — no image ever leaves your device

### MODULE 03 — Camera Lab
- Live camera with full HUD overlay
- Four corner bracket viewfinder
- Face alignment oval guide (pulses for ID photos)
- LIVE indicator
- 8 real-time filters: Normal · B&W · Sepia · Vivid · Soft · Hue+ · Pop · Matte
- 4 live adjustment sliders: Brightness · Contrast · Saturation · Blur
- Front / rear camera switch
- Mirror / flip toggle
- Snap outputs directly to preset size (SA ID, Passport, Square, Native)
- Filter baked into exported photo
